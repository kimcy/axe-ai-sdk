# 영속화 (Persistence)

`persistence` 옵션을 켜면 메시지를 `localStorage` (또는 커스텀 스토리지) 에
자동 저장합니다.

```tsx
useChat({
  transport,
  persistence: {
    key: 'chat-demo',
  },
})
```

## 안전한 저장

저장 직전에 **진행 중 상태(`pending` / `streaming`)를 정리**합니다. 이렇게
하면 사용자가 스트리밍 도중 새로고침하더라도, 반쯤 완성된 메시지가 `streaming`
상태로 영원히 남는 일이 발생하지 않습니다.

구체적으로:

- 마지막 메시지가 `pending` 상태면 제거
- `streaming` 상태면 `aborted` 로 바꿔서 저장

## 커스텀 스토리지

`localStorage` 대신 다른 저장소를 쓰려면:

```tsx
useChat({
  transport,
  persistence: {
    key: 'chat-demo',
    storage: {
      getItem: (key) => myStore.get(key),
      setItem: (key, value) => myStore.set(key, value),
      removeItem: (key) => myStore.delete(key),
    },
  },
})
```

## 직렬화

기본 구현은 `JSON.stringify` 를 사용합니다. 사용자 정의 `metadata` 에 직렬화
불가능한 값(함수, `BigInt` 등)을 넣지 마세요.
