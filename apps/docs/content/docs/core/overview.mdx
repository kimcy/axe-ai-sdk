# @axe-ai-sdk/core

전송 계층에 의존하지 않는 순수한 스트리밍 챗 코어입니다. 런타임 의존성이
없으며 브라우저·Node·엣지 런타임 모두에서 동작합니다.

## 내보내는 것

```ts
import {
  // 타입
  type Message,
  type MessageRole,
  type MessageStatus,
  type StreamPart,
  type ChatRequest,
  type ChatTransport,
  type ThinkingStep,
  type ToolCall,
  type Citation,
  type ControllerStatus,

  // 런타임
  ChatController,
  parseSSE,
  createId,

  // 에러
  ChatError,
  TransportError,
  IdleTimeoutError,
} from '@axe-ai-sdk/core'
```

## 번들 크기

- `@axe-ai-sdk/core` — 약 11 KB (minified, pre-gzip)
- 의존성 — 없음
