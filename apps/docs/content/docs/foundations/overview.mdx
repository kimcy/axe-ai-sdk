# 기본 개념

axe-ai-sdk 는 세 개의 계층으로 구성됩니다.

```
┌──────────────────────────┐
│   useChat (React)        │  ← UI 바인딩, 입력 상태, 낙관적 렌더링
├──────────────────────────┤
│   ChatController         │  ← 상태 기계, 요청 격리, 아이들 타임아웃
├──────────────────────────┤
│   ChatTransport          │  ← 백엔드 어댑터 (직접 구현)
└──────────────────────────┘
```

- **Transport** 는 "어떻게 서버와 말하는가" 를 담당합니다. OpenAI, Anthropic,
  MCP, 사내 게이트웨이 중 어느 것이든 동일한 인터페이스를 구현합니다.
- **Controller** 는 "무엇이 진행 중인가" 를 관리합니다. 메시지 배열을 유지하고,
  중단/재전송/에러 상태를 처리합니다.
- **useChat** 은 컨트롤러를 React 렌더 트리에 연결합니다.

각 계층은 다음 섹션에서 자세히 다룹니다.

- [스트리밍과 SSE](/docs/foundations/streaming)
- [전송 계층](/docs/foundations/transports)
- [메시지 상태 기계](/docs/foundations/state-machine)
